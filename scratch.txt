In Progress® Version 6.0 or later, the ANSI SQL (-Q) startup parameter allows you to redefine the semicolon as a terminator. This startup parameter enforces strict ANSI SQL conformance and allows you to terminate SQL statements with a semicolon. The ANSI SQL (-Q) parameter allows OpenEdge® to run standard SQL statements and scripts built with other products.\n\nThe ANSI SQL (-Q) parameter disables the use of the semicolon within UNIX escapes.
